1장. 변수 

(1) 기초 데이터 타입 

1. 논리형
boolean: true, false의 두가지 값을 가짐 

2. 문자형
char: 한 글자의 문자를 저장하는 데이터 타입 

3. 정수형
byte (-128~127의 정수) - 1byte
short(대략 -32,000~33,000의 정수) 
int (대략 -20억 ~ 21억의 정수) -> default 데이터 타입으로, 따로 명시를 해주지 않는 한 정수는 int 타입으로 저장됨 -4byte 
long(그 이상의 값들이 저장됨)  -8byte

3. 실수형 
float (-3.4E38 ~ +3.4E38의 실수)
double (1.7E308 ~ + 1.7E308의 실수) -> 실수의 default 데이터 타입으로 따로 명시를 해주지 않으면 실수는 double 타입으로 저장됨  -8byte

(2) 참조 데이터 타입 

기초 데이터 타입과 달리 변수 선언을 하면 변수에는 실제 데이터가 담긴 heap메모리 주소값을 담는다. 

변수 타입의 이름이 대문자로 시작함

(3) 형 변환 
1. 묵시적 형변환: 작은 byte의 데이터 타입을 더 큰 byte의 데이터 타입으로 변환함 
2. 명시적 형변환: 큰 byte의 데이터 타입을 더 작은 byte의 데이터 타입으로 변환함
