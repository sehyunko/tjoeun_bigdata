2장. 조건문 

1. if문 

if(조건1){
	수행할 코드1
} else if(조건2) {
	수행할 코드2
} else(조건1, 조건2도 아닌 나머지 경우) {
	수행할 코드
}

2. switch문 

switch(조건. switch문의 조건에는 정수, 문자, 문자열만 기입할 수 있음){
	case 조건1:
		수행할 코드1
	break; (break를 선언 해 주지 않으면 case 2로 넘어 가게 됨) 
	case 조건2:
		수행할 코드2
	break; 
}

3. while문 (if문 과는 다르게 정확히 수행할 횟수가 몇번 일지 모르고 종료 조건은 알 경우 사용) 

while(실행조건. true/false 값을 기입하며 false가 될 경우 while문을 종료하게 된다) { 
	수행할 코드 
} 

4. do-while문 (while문과 비슷하지만 무조건 첫번쨰 실행은 수행한다) 

do{
	수행할 코드 

} while(실행 조건 true/false);
