04. CSS 선택자

body 부분에 작성된 태그들 중에 원하는 태그를 골라서 stylesheet에서 작업을 하는데에 CSS 속성자가 사용된다. 

1. 전체선택자 (*) --> 페이지 초기화

페이지 전체를 불러온다. 보통 body에 초기값으로 지정된 margin이나 padding을 원하는값으로 조정하거나, 아예 없애고 싶을때 사용한다. 

2. id 선택자 (#)과 class 선택자 (.) 

(1) #
원하는 div를 불러오고 싶을때, #div의 아이디 형태로 불러온다. 

(*구문작성시 주의할 것: 예를 들어, 

<div id="header">
	<div id="logo">
		<ul>
		   <li>메뉴1<li/>
		   <li>메뉴2<li/>
	               <li>메뉴3<li/>
		<ul/>
	<div/>
<div/>

위의 코드에서 logo 태그 안에 담긴 unordered list를 불러와 stylesheet에서 작업을 하고 싶다면, 

#header #logo ul 을 통해 선택자를 이용해 불러오는데, header와 ,logo 사이에 띄어쓰기를 하지 않으면 오류가 남에 꼭 주의할 것.)

(2) .

클래스 선택자는 id와 쓰이는 방법은 동일하다. 


3. 속성 선택자 

tag의 속성을 선택하여 원하는 태그만 가져오고 싶을때 사용. 

선택자[속성]: [ ]내의 속성을 가진 태그만 가지고 옴. 
선택자[속성=문자]: 속성이 지정한 문자롸 같은 속성을 가진 태그만 가지고 옴. 
선택자[속성^=문자]: 속성이 지정한 문자로 시작하는 속성을 가진 태그만 가지고 옴. 
선택자[속성$=문자]: 속성이 지정한 문자로 끝나는 속성을 가진 태그만 가지고 옴. 


4. 자손선택자(>), 후손선택자

div 아래에 여러 자식 div들이 포함되어있는 형태일때,

자손선택자는 직속 하위, 또는 바로 아래 태그를 선택해서 가져온다. 

반면 후손 선택자는, 몇번째 하위 자손인지에 구애 받지 않고 하위 자손 태그를 불러온다. 


5. 동위 선택자

div 공간 분할 태그들이 서로 종속 되있거나 한 관계가 아닐때, 

+ div 선택자: 자기자신 바로 다음에 이어지는 div 태그를 불러옴.
~ div선택자: 자기자신 다음에 이어지는 모든 div 태그를 불러옴.
