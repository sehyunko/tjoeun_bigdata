7장. 제어문 

1. if 문 

자바, R에서와의 if 문과 동일 

<코드 작성 구문> 

if 조건1:
    수행문 
elif 조건2:
    수행문 
elif 조건3: 
    수행문3
else: 
    수행문4 

2. for 문 

(1) 수행할 반복횟수를 지정하는 경우 

range( ) 함수를 통해 반복횟수를 지정. 어떻게 보면 range( ) 함수도 특정 범위 내의 숫자를 담은 list를 반환 하므로 (1)과 (2)가 같은 방법이라고 볼 수 있다. 


(2) iterable 객체를 iterate 하는 경우 

for c in iterable객체 -> iterable한 객체내의 원소를 하나 하나 돌며 수행문을 수행한다. 

3. while문 

while 문 또한 자바, R 에서의 while문과 동일 

while 조건: 
     수행 문장
else: 
     수행 문장

3. 중첩루프 탈출 

1. break, continue를 사용하여 탈출한다. 

자바, R에서의 break, continue 문과 기능은 동일하다. 

아래의 예외구문을 이용한 중첩루프 탈출과는 달리 break를 이용하면 해당 루프만 빠져나오기 때문에 여러번의 if-break 문을 통해 바깥루프를 하나하나씩 빠져 나와야 한다.

*break: 만나면 바로 해당 루프를 빠져나온다 

*continue: 보통은 if 문과 함께 쓰는데, 일정한 조건에따라 continue를 만나면 다시 for문의 맨 처음으로 돌아간다. 

2. 예외구문을 이용한다. 

if 문을 통해 일정한 조건시, raise 문으로 예외를 발생시켜 중첩루트를 아예 다 빠져나와 except 절로 빠지게 할 수 있다.
