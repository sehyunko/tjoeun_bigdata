(1) 두 배열 분리하기 

np.hsplit( 분리시킬 배열, 분리갯수) : 분리 갯수 대로 동등한 갯수 대로 배열을 분리. 세로축을 기준으로 분리한다 ( stack 함수와 반대로 hsplit이 세로 축을 기준으로 적용된다)

np.hsplit( 분리시킬 배열, [분리 기준 인덱스] ) : 스칼라 값이 아닌 배열에 넣어주면, 분리할 갯수가 아니라 분리 기준 인덱스를 주게 된다. 

np.vsplit(  이하동일 ) : hsplit 과 반대로 가로축을 기준으로 배열을 분리한다. 

np.dsplit( 이하동일 ) : 면을 기준으로 분리 하게 된다. 

np.split( 이하동일, axis=0 or 1 or 2) : axis=0이면 vsplit, axis=1이면 hsplit, axis=2 이면 dsplit이 된다. 

--> 위의 함수들은, 분리갯수를 인자값으로 주어주면, 분리 결과가 균일 하게 딱 맞아 떨어져야 오류가 안난다. 예를 들어, 10열 짜리 배열 3개로 분리하려 하면 3,3,4 개의 배열로 균일하게 안 분리 되므로 오류가 발생한다. 

array_split( 분리시킬 배열, [분할기준 인덱스] or 분할 갯수,  axis=?) : np.split 함수와 동일하나, np.split과 달리 분할 갯수를 주어 줬을때, 분할 결과가 균일하게 딱 맞아 떨어지지 않아도 분리가 된다.
