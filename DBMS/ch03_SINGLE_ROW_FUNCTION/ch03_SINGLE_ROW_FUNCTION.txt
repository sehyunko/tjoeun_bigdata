3장. 단일행 함수 

그룹함수 또는 다중행 함수가 다중행에 적용되어 결과 값은 단일행인 반면 단일행 함수는 결과값이 다중행으로 나온다. 

1. 숫자함수 

TRUNC(X,N) : X를 N의 자리에서 버림. N에 양수 값을 입력하게 되면 소수자리 숫자를 의미하고, 정수 자리를 입력하고 싶다면 음수를 입력한다 
	    예를들어, N이 2라면 소수점 2번째 자리에서 버림, N이 -2라면 10의자리에서 버림을 수행 

MOD(X,N)   : X를 N으로 나눈 나머지 값을 리턴 

ROUND(X,N) : X를 N의 자리에서 반올림한다. N에 대한 입력값은 TRUNC 함수와 동일하게 적용된다. 음수, 양수 구분 


2. 문자 처리함수 
LOWER(STR), UPPER(STR) -> 문자를 각각 소문자, 대문자로 바꿈 

INITCAP(STR) -> 문자의 첫글자만 대문자, 나머지는 소문자로 반환 

CONCAT(STR1,STR2) -> 두 문자를 합침. 주의할 것은 3개이상의 인자값은 받지 않는다. 3개이상은 || 연산자 이용 

TRIM(STR) -> 문자의 양끝 공백을 제거 

SUBSTR(STR, 시작위피, 추출 갯수) -> 문자의 일부를 추출

LENGTH(STR) -> 문자의 길이를 반환 

REPLACE(바꿀문자, 바뀌어질 문자) -> 문자를 변환 

3. 날짜 함수

SYSDATE -> 현재 시간을 반환한다 

MONTHS_BTWEEN -> 두 날짜 사이의 달 수를 반환한다 (어느 달은 30일, 어느 달은 31일인데 그런 것을 고려해서 계산한다) 

ROUND(날짜, '단위') -> 날짜를 반올림 한다 (예를 들어 'MONTH'를 넣으면 15일 기준으로, 'YEAR'를 넣으면 6월을 기준으로 반올림 한다) 

EXTRACT(YEAR/MONTH FROM 변수) -> 날짜 데이터에서 연도, 달 등을 추출한다

4. 형변환 함수 

TO_CHAR(날짜데이터, '포맷') -> 날짜 데이터의 연도, 달, 일자 등을 추출할 때 유용하게 쓰인다

TO_DATE('문자','FORMAT') -> 시스템마다 날짜를 표기하는 법이 다르므로, TO_DATE 함수를 이용하여 날짜 연산을 하는것이 안전 하다

TO_NUMBER('문자형 숫자','포맷') -> '99,999', '00,000' 9로 채우면 숫자가 없는 자리의 숫자는 풀력을 안하는 반면, 0으로 채우면 숫자가 없는 자리의 숫자는 0으로 대신 채워서 출력한다
