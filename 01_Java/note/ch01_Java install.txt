1장. Java 설치 및 기본 개념 

**프로그램 언어의 종류 **

1. CPU 종속적 언어: 컴파일 된 파일이 특정 CPU 에서만 작동함 
ex. 어셈블리어 

2. OS 종속적 언어: 컴파일 된 파일이 특정 OS (윈도우, 리눅스, 맥 등등) 에서만 작동 함. 
ex. C, C++ 

3. OS 독립적 언어: 컴파일 된 파일이 어느 CPU, OS든지 모두 작동 가능함 
ex. java 

**자바 설치** 

1.자신의 pc에 맞는 자바 프로그램을 설치한다. (32비트(X86) 인지 64비트(X64) pc인지 확인 할 것) 

2.사용목적에 맞는 자바 버전을 설치해야 한다. (Hadoop을 사용하기 위해선 자바 8, Eclipse 4.16버전 설치 필요) 
 
3.설치 후 javac 파일이 내 컴퓨터 내 어느 위치든 접근이 가능 하도록 환경변수를 설정해준다. 
(Java JDK가 있는 파일 경로를 JAVA_HOME 변수 담고 시스템 변수에 추가  -> 사용자 변수인 Path에 ;%JAVA_HOME%\bin를 추가해준다)
