7장. 추상 클래스 

1. 추상 메소드 (abstract) 

특정한 메소드를 부모 클래스에서 정의만 해놓고 상속 받는 자식 클래스에서 오버라이딩을 하도록 강요하는것 
-> 추상 메소드를 하나 이상 포함한 클래스를 추상 클래스라 부른다. abstract 이라는 예약어를 통해 정의

public abstract class 클래스 이름{
	public void abstract 메소드이름( ){
		오버라이드
	}
}

2. final 제한자 

abstract와는 반대의 개념. 

맴버 변수나 메소드를 final 제한자를 통해 정의를 해 놓으면, 맴버 변수의 경우 값 변경이 불가하고, 
메소드의 경우 오버라이딩이 불가하다. 

주로 상수를 정의 할때 final 제한자와 함께 static 변수로 선언한다.
