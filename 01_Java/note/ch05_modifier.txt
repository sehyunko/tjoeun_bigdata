5장. 접근 제한자 

1. 접근 제한자의 종류 
public 다른 모든 클래스에서 접근이 가능 
protected 동일 패키지내 클래스나 상속받은 클래스일 경우 접근이 가능 
default(생성자를 따로 명시해 주지 않는 경우) 동일한 패키지 내 클래스들에 한해서 접근 가능 
private 해당 클래스내에서만 접근이 가능함 

2. 맴버 변수의 선언 
[접근 제한자] [static/final] 데이터타입 변수명; 

static: 클래스로부터 여러 개의 객체를 생성해도 초기에 생성했던 객체 하나만 생성이 됨. 한 클래스로부터 생성된 static 변수는 
        모든 객체들 사이에서 공유하게됨. static 맴버 변수의 경우 객체를 따로 생성하지 않고, 클래스 이름을 통해 바로 접근한다. 
final: 이후 변경이 불가한 상수 값을 정의할 때 명시한다. 보통 상수(종단 변수) 값은 따로 클래스를 만들어 모아 보관하여 유지보수가 
       용이하게 끔 함.
