8장. 인터페이스 (Interface) 

추상 클래스와 비슷한 개념으로, 인터페이스를 구현할 클래스에서 작성할 추상 메소드, 상수 값만을 정의해 놓는다.
-> 추상 클래스와는 달리 일반 메소드, 변수는 올 수 없다. 오로지 추상메소드, 상수 값만을 작성 가능함. 
-> 현 Java에서는 여러 상위클래스를 다중상속 할 수 없지만, 인터페이스의 경우 여러 인터페이스를 다중 구현 가능하다 

1. 작성 구문 

public interdace 인터페이스명 {
	public int MAX_VOLUME = 10; //인터페이스 내 작성 하는 상수값을 담는 변수는 대문자로 표기한다
	
	public abstract void 추상메소드명( ); 
} 

인터페이스를 구현하는 클래스를 작성하는 구문 

public class 클래스명 implements 인터페이스1,인터페이스2....{
	 생성자 

	 인터페이스 내 추상 메소드 오버라이드 
	
	 setter, getter
}
