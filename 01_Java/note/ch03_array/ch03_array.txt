3장. 배열 

배열: 동일한 자료형의 집합. ArrayList와는 다르게 한번 배열의 크기를 선언하면 변경이 불가함. 

1. 배열 변수 선언 방법 

int[] array1 = new int[6]; 배열의 크기만을 선언해 놓는 방법 (이 경우 배열 내의 값들은 0으로 초기화된다)

int[] array2 ={0,1,2,3,4,}; 베열의 크기는 물론, 입력값도 같이 선언하는 방법 

** 배열이 담긴 변수는 레퍼런스 변수임. 변수안에 직접적인 데이터가 담긴게 아닌 데이터가 담긴 메모리의 주소가 담겨있다. 

2. 배열 내 데이터 접근 방법 

(1) 일반 for문 이용

for(int i=0; i<array.length; i++){  //변수의 인덱스 값은 0부터 시작함에 주의
	sysout(array[i]); 
} 

(2) 확장 for문 이용 

for(int a:array){
	sysout(array[i]); 
} 

3. 배열 복사하기 

(1) for문을 이용해 데이터 접근 후 복사하는 방법 

(2) System.arraycopy(원본 배열 객체, int 원본 시작 인덱스, 복사본 배열객체, int 복사본 시작 인덱스, int 복사 길이)
