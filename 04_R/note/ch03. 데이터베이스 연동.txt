3장. 데이터 베이스 연동 

먼저 자바의 환경변수를 설정해놓고, 자바 jre 폴더 > lib > > ext 폴더에 있는 로드 드라이버인 ojdbc6.jar 파일을 내 R working directory에 복사해 넣는다. 


1. Oracle DB와 연동 하는법 : 사전 준비 -> sqldf 패키지와 RJDBC 패키지를 로드한다. 

(1) 드라이버 클래스 로드 



(2) 데이터 베이스 연결 

(3) SQL문 전송 + 결과 받기 

** SQL문 전송과 결과 받기를 한번에 하기 : dbGetQuery(conn객체, 쿼리문) 을 사용하면 된다. 

(4) DB 데이터 수정 쿼리문 전송 (update, insert, delete)

(5) 데이터 연결 해제 

(6) 드라이버 언로드 -> 보통 데이터 연결을 해제 하는 순간 드라이버도 같이 언로드 된다.



2.  MySQL DB과 연동 하는법 -> RJDBC와 RMySQL을 로드 한다.

(1) 데이터 베이스 연결

(2) SQL문 전송 + 결과받기 

** 오라클 DB와 마찬가지로 dbGetQuery( ) 함수를 이용하여 쿼리문 전송과 결과받기를 한번에 수행할 수 있다. 

** dfReadTable(conn 객체, "테이블 명칭") 을 통해 테이블 전체 데이터를 한꺼번에 조회할 수도 있다. 

(3) 데이터 베이스 연결 해제
