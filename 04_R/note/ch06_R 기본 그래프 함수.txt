6장. R 기본 그래프 함수  (base graphing system) 

(1) 고수준 그래프 함수 : 실행 할 떄 마다 새로운 캔버스를 얻어 그린다.  

ex) barplot( 인자값으로 table 자료만 받음 ),  boxplot( ), hist( ), pie( ) 등등  


(2) 저수준 그래프 함수 : 실행을 하면 전 단계에서 고수준 그래프 함수를 통해 그려진 그래프의 위에 그린다.   

ex) lines( ), abline( ), point( ), text( ) 등등 

text 함수에서 adj 옵션 -> 글자를 하나의 박스로 두르고 7시 방향이 (0,0), 1시 방향이 (1,1) 이라고 보고 위치를 조정한다. 

이후 ggplot에서의 vjust, hjust 또한 동일한 원리를 적용해 글자 위치를 조정한다. 


(3) par( )함수: 전체적인 캔버스의 설정을 지정한다. 함수를 실행시키면, 캔버스의 설정을 지정하는 동시에 반환값으로 직전의 캔버스의 설정을 반환한다. 

-> mfrow 옵션: 벡터의 형태로 c( 행의 수  , 열의 수 )를 주어주면 한 캔버스 안에 여러 개의 subplot을 그린다. 

-> bty 옵션: 그래프 테두리의 모양을 지정한다. 가능한 값은 1,7,c,u,],o 가 있다. 

-> new = TRUE/FALSE 옵션: 고수준 함수 위에 고수준 함수를 덮어 쓸 여부를 결정한다. 

-> mar 옵션: 하,좌,상,우의 마진을 결정한다. 

 
** par( ) 함수로 지정한 설정은 ggplot2의 함수들에는 적용이 안된다.
